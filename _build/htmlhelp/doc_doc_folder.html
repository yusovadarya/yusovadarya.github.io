Папки документов¶

Создать папку¶

  1. Нажмите кнопку в правом нижнем углу страницы и выберите пункт Папка.

  1. Ваша папка появилась в списке документов. Введите имя новой папки. Оно сохранится автоматически.

Изменить название папки¶

  1. Выберите папку, название которой нужно изменить, и наведите на нее курсор.

  2. Нажмите кнопку ⋮ справа.

  3. Выберите пункт Переименовать.

  4. Название папки будет выделено цветом. Введите новое название и нажмите вне строки.

Название сохранится автоматически.

Удалить папку¶

  1. Выберите папку, которую нужно удалить, и наведите на нее курсор.

  2. Нажмите кнопку ⋮ справа.

  3. Выберите пункт Удалить.

  4. Подтвердите действие.
